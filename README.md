# 🚀 TaskHub - Premium Task Manager App

TaskHub adalah aplikasi manajemen tugas kelas berat berbasis Android dengan sistem **Kanban Board** (*Swipeable Tabs*). Aplikasi ini dirancang dan dikembangkan secara *Native* (Java) untuk memenuhi standar "Aplikasi Kompleks" dengan menggabungkan berbagai materi *Mobile Programming* ke dalam satu ekosistem yang utuh dan fungsional.

## ✨ Fitur Unggulan
* **Sistem Kanban Board:** Manajemen tugas interaktif yang dibagi menjadi tab Belum, Proses, dan Selesai.
* **Smart Pengingat (AlarmManager):** Notifikasi pintar H-1 lengkap dengan pola getaran dan audio *custom*.
* **Aksesibilitas & Suara:** Dukungan *Text-to-Speech* untuk membacakan tugas dan *Speech Recognition* untuk input tugas *hands-free*.
* **Sensor Hardware (Shake to Prioritize):** Menggunakan *Accelerometer* untuk merekomendasikan tugas acak hanya dengan menggoyangkan HP.
* **Multimedia & Storage:** Integrasi *Camera Intent* untuk lampiran gambar tugas.
* **Kolaborasi (Share Intent):** *Generate* daftar tugas ke dalam format `.txt` dan bagikan langsung ke WhatsApp.
* **Penyimpanan Lokal (SQLite):** Database cepat, aman, dan permanen.

## 🎨 UI/UX Design
Mengusung tema **Aurora Misty Forest** (*Premium Dark Mode*) dengan elemen *Glassmorphism* dan standar aksesibilitas kontras warna yang nyaman di mata.
